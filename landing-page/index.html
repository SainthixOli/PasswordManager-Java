Arquitetura & Stack

Construído sobre pilares de segurança e performance com Java 21.

Core Backend

Spring Boot 3.2 rodando em Java 21 LTS. Virtual Threads e Records para máxima eficiência.

JCA Security

Java Cryptography Architecture. Hash SHA-512 (Auth) e AES-256 (Vault) nativos.

Secure Persistence

Armazenamento criptografado em disco. Zero dependência de SGBDs externos para portabilidade.

SSR Frontend

Thymeleaf + CSS Moderno. Renderização segura no servidor sem exposição de lógica crítica.

Deep Dive

Exploração técnica dos módulos do sistema.

Módulo de Autenticação

Sistema de verificação de identidade Zero-Knowledge (parcial). Apenas o hash salted é persistido.

  • Proteção contra Rainbow Tables via Salt Dinâmico
  • Design "True Black" para conforto visual

Vault Engine (AES-256)

Núcleo criptográfico. Cifragem simétrica de grau militar para cada credencial armazenada.

  • Operações em memória segura
  • Mascaramento de dados sensíveis na UI

Gerador de Entropia

Utilitário para maximização de segurança de credenciais externas.

  • Alta variância de caracteres
  • Comprimento customizável até 256 bits efetivos
